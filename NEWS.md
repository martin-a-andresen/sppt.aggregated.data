# sppt.aggregated.data 0.1.0

## Initial Release

* First release of sppt.aggregated.data package
* Core function `sppt()` for spatial pattern point tests
* Bootstrap resampling for confidence intervals
* S-Index and Robust S-Index metrics
* Support for multiple export formats (shp, csv, txt, rds, gpkg)
* Map generation and export for bivariate comparisons
* Option to work with percentages or counts
* Fixed base variable option for specific comparison scenarios
* Comprehensive vignette with detailed examples and best practices
